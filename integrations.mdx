## Overview

Connect BG Ventures to your favorite tools to automate workflows, receive notifications, and embed external content directly into your documentation. Use built-in integrations for popular apps, set up webhooks for real-time updates, leverage the REST API for custom solutions, or embed resources from other services.

<Callout kind="tip">
  Start with popular integrations for quick wins, then explore webhooks and API for advanced automation.
</Callout>

## Setting Up Webhooks

Webhooks notify external services about events in BG Ventures, such as document updates or new versions. Follow these steps to configure one.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to your project settings > Integrations > Webhooks. Click "New Webhook".
  </Step>
  <Step title="Configure Events" icon="zap">
    Select events like `document.updated` or `page.published`. Enter your endpoint URL.
  </Step>
  <Step title="Add Secret" icon="lock">
    Generate and copy the signing secret for verification.
  </Step>
  <Step title="Test" icon="play">
    Trigger a test event and verify the payload at your endpoint.
  </Step>
</Steps>

Example webhook payload:

<CodeGroup tabs="JSON">
  ```json
  {
    "event": "document.updated",
    "project_id": "proj_123abc",
    "document_id": "doc_456def",
    "timestamp": 1699123456
  }
  ```
</CodeGroup>

## Custom API Integrations

Build tailored integrations using the BG Ventures REST API at `https://api.example.com/v1`. Authenticate with your API key.

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Authorization: Bearer YOUR_API_KEY`
</ParamField>

<ParamField query="project_id" param-type="string" required="true">
  Your project identifier.
</ParamField>

Example: List integrations.

<Request tabs="JavaScript,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/integrations?project_id=proj_123', {
    headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
  });
  const data = await response.json();
  ```
  ```bash
  curl -H "Authorization: Bearer YOUR_API_KEY" \
       "https://api.example.com/v1/integrations?project_id=proj_123"
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "integrations": [
      {
        "id": "int_789ghi",
        "type": "slack",
        "status": "active"
      }
    ]
  }
  ```
</Response>

## Embedding External Resources

Embed content from third-party services to enrich your docs.

<Tabs>
  <Tab title="Iframe" icon="layout">
    Use secure iframes for dashboards or tools.
    
    ```
    <iframe src="https://dashboard.example.com/embed" width="100%" height="400"></iframe>
    ```
  </Tab>
  <Tab title="API Data" icon="database">
    Fetch and display live data.
    
    <CodeGroup tabs="JavaScript">
      ```javascript
      fetch('https://api.example.com/v1/data?key=YOUR_API_KEY')
        .then(res => res.json())
        .then(data => document.getElementById('embed').innerHTML = JSON.stringify(data));
      ```
    </CodeGroup>
  </Tab>
</Tabs>

<Callout kind="alert">
  Always validate external content for security. Use `sandbox` attributes on iframes.
</Callout>

## Overview

BG Ventures empowers you to create, organize, and maintain comprehensive documentation for your projects. You get a centralized space to manage Markdown-based docs, collaborate with teams, and publish polished documentation sites. Focus on content while the platform handles versioning, search, and deployment.

This documentation space serves as your hub for all BG Ventures features. Explore key capabilities, set up your first project, and customize your docs to match your brand.

<Callout kind="info">
BG Ventures uses brand color `#3B82F6` for primary elements like buttons and links.
</Callout>

## Brand Guidelines

Follow these guidelines to maintain consistency across your documentation.

<Expandable title="Primary Color Usage" default-open="true">
Use `#3B82F6` for:

- Primary buttons and calls-to-action
- Links and hover states
- Header accents

| Element       | Hex Code   | Usage Example                  |
|---------------|------------|--------------------------------|
| Primary      | `#3B82F6` | Buttons, links                |
| Secondary    | `#1E40AF` | Subtle backgrounds            |
| Text         | `#1F2937` | Body text                     |

Avoid using the primary color for text on light backgrounds to ensure readability.
</Expandable>

## Quick Start

Get up and running with BG Ventures in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit [https://app.bgventures.com](https://app.bgventures.com) and create a free account. Verify your email to unlock full features.
  </Step>

  <Step title="Create a Project" icon="plus">
    Click "New Project" in the dashboard. Enter a name like "My App Docs" and select a template.
  </Step>

  <Step title="Add Your First Doc" icon="file-text">
    Create `introduction.mdx` with basic Markdown.

````mdx
# Welcome

This is your first document.
````

  </Step>

  <Step title="Publish" icon="globe">
    Click "Publish" to generate your live site at `your-project.bgventures.com`.
  </Step>
</Steps>

<Tabs>
  <Tab title="npm Integration" icon="package">
    Embed docs in your app:

````javascript
import { DocsEmbed } from '@bgventures/embed';

<DocsEmbed projectId="your-project-id" />
````

  </Tab>

  <Tab title="GitHub Sync" icon="github">
    Connect your repo for auto-sync:

````bash
git remote add bgventures https://git.bgventures.com/your-project.git
git push bgventures main
````

  </Tab>
</Tabs>

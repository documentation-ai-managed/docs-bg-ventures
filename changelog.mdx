## Upgrade Guide

Follow these steps to upgrade BG Ventures smoothly.

<Steps>
  <Step title="Check Compatibility" icon="shield">

    Review your current configuration against release notes.

  </Step>
  <Step title="Backup Data" icon="database">

    Export your projects to avoid data loss.

    <CodeGroup tabs="CLI,Dashboard">
      ```bash
      bg-ventures export --project all --output backup.json
      ```
      ```javascript
      // Dashboard API call
      const response = await fetch('https://api.example.com/v1/export', {
        method: 'POST',
        headers: { Authorization: `Bearer ${YOUR_TOKEN}` },
        body: JSON.stringify({ project: 'all' })
      });
      ```
    </CodeGroup>

  </Step>
  <Step title="Update Client" icon="download">

    Install the latest version.

    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm update bg-ventures
        ```
      </Tab>
      <Tab title="Docker" icon="docker">
        ```bash
        docker pull bgventures:latest
        ```
      </Tab>
    </Tabs>

  </Step>
  <Step title="Verify" icon="check-circle">

    Test key workflows and restore from backup if needed.

  </Step>
</Steps>
